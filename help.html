Introduction

BATT is a simple tool to help plot out all the annual Burger Love burgers on a map, find the ones that are near to you, find ones that might interest you, and keep track of what ones you are interested in or have eaten. It gives you a few basic tools to create and update a "Burger Profile" that describes exactly what your Burger Love plans and interests are.

Click on any icon you see to check out the burger info! Icons containing numbers are there to cluster together burgers that are close together. Click on a cluster to center it in the map window.

Burger Markers

Both marker icons and status buttons are colour-coded so you can see their status at a glance. They look like this:

Icon Button Colour Description
No burger setting has been applied yet. Once a setting is applied, you can't revert to this.
You won't be having this burger.
You'd like to have this burger in the future.
You've had this burger already.

Click an individual marker on the map to bring up the burger info window for that restaurant.

Clicking on the button will give you the option to set the status of the burger. Once set, the status will be saved, and will be remembered when you come back to the site using the same browser.

Filtering markers

Clicking on the "Filter Markers" menu option at the top of the screen brings up a list of filters you can set. Their status is designated by two icons:

Unset
Indicates a filter that has not been set. Burgers with this status will display on the map.
Set
Indicates a filter that has been set. Burgers with this status will not display on the map.

Markers representing restaurants that are currently closed can be filtered out by clicking "Closed Restaurants". A window will pop up allowing you to offset this filter by a number of hours, so that you can filter out, for example, restaurants closing within the next hour or two.

Use the "Disable All Filters" option to unset all filters and show all burgers on the map.

Search (experimental!)

Search is the current development focus, so consider it in beta, I guess?

Search results cycle, so clicking "Go" or hitting "Enter" multiple times will rotate you through the results list.

The search is pretty lazy; you don't have to capitalize, but it doesn't catch missing punctuation at the moment, so "hunter's" would work, but not "Hunters". This is on the TODO list.

Importing and exporting burger profiles

Burger profiles, which comprise your current burger and filter settings, are stored per-browser per-machine, which might not be convenient for you. Importing and exporting profiles allows you to carry them over to another computer or browser.

Clicking the "Tools" menu at the top shows two buttons you can use to do this:

Import Burger Profile
Opens up a load screen to allow you to input a previously-exported burger profile. This file must be a valid JSON type file.
Export Burger Profile
Saves the current burger profile to burger_profile.json.

A button also exists in that menu, "Reset Burger Profile", to completely wipe the current burger profile. You'll be prompted if you want to do so.
